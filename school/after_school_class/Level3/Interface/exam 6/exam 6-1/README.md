# 서브 문제

```
PayMain에서 결제 수단과 금액을 모두 입력 가능하게 만드시오.
(단, 결제 수단에 ‘exit’가 입력되면 프로그램을 종료함.)
```
```
// 출력 예시
결제 수단을 입력하세요:kakao
결제 금액을 입력하세요:5000
결제를 시작합니다: option = kakao, amout = 5000
카카오페이 시스템과 연결합니다.
5000원 결제를 시도합니다.
결제가 생공했습니다.
결제 수단을 입력하세요:hahaha
결제 금액을 입력하세요:10000
결제를 시작합니다: option = hahaha, amout = 10000
결제 수단이 없습니다.
결제가 실패했습니다.
결제 수단을 입력하세요:exit
프로그램을 종료합니다
```
